## راهنمای ابزارها

هر راهنما یک برگه را قدم به قدم توضیح می‌دهد:

| راهنما | توضیح |
|---|---|
| [اسکنر اندپوینت](endpoint-scanner.md) | پیدا کردن اندپوینت‌های واپ کلاودفلر با کانفیگ WireGuard |
| [اسکنر آی‌پی (تمیز)](ip-scanner.md) | اسکن محدوده آی‌پی‌های کلاودفلر برای پروکسی‌های تمیز |
| [جایگزین آی‌پی](ip-replacer.md) | جایگزینی IP:port در کانفیگ‌های اشتراک با اندپوینت‌های تمیز |

## ارتباط ابزارها

```
اسکنر اندپوینت  ──>  انتخاب سریع‌ترین اندپوینت واپ
                           │
اسکنر آی‌پی     ──>  پیدا کردن آی‌پی‌های پروکسی تمیز کلاودفلر
                           │
جایگزین آی‌پی    ──>  جایگزینی IP:port در کانفیگ‌ها با آی‌پی‌های تمیز
```
