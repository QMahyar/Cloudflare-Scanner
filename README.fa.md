# Cloudflare Scanner

**اسکنر اندپوینت و آی‌پی واپ کلاودفلر** — یک ابزار دسکتاپ ویندوز که محدوده آی‌پی‌های کلاودفلر را اسکن می‌کند تا اندپوینت‌های working واپ را پیدا کند. از [xray-core](https://github.com/XTLS/Xray-core) برای WireGuard با نویز UDP استفاده می‌کند — زمانی مفید است که ISP شما ترافیک استاندارد واپ را مسدود کرده باشد.

> [English version](README.md)

## ویژگی‌ها

- **اسکن اندپوینت** — تولید تصادفی اندپوینت‌های IPv4/IPv6 کلاودفلر و تست هرکدام
- **نویز UDP** — padding تصادفی + تأخیر برای عبور از مسدودسازی DPI
- **نتایج زنده** — اندپوینت‌های working هم‌زمان با قبول شدن handshake نمایش داده می‌شوند
- **اعمال دسته‌جمعی** — یک اندپوینت working را انتخاب کنید و به چندین فایل کانفیگ هم‌زمان اعمال کنید
- **رابط وب** — رابط کاربری تمیز تحت مرورگر که توسط خود برنامه سرو می‌شود، بدون نیاز به وابستگی اضافی
- **خودکفا** — دانلود شامل xray-core است، نیاز به تنظیمات اضافی ندارد
- **قابل لغو** — اسکن را هر لحظه می‌توانید متوقف، دوباره اجرا یا ریست کنید

## نحوه کار

1. فایل `.conf` وایرگارد خود را انتخاب کنید (استاندارد واپ یا سبک هاگوارتز با فیلدهای اضافی)
2. تعداد اسکن، ورژن آی‌پی (IPv4/IPv6/هر دو) و نویز UDP را تنظیم کنید
3. روی **Start Scan** کلیک کنید — سرور اندپوینت‌ها را به صورت موازی با استفاده از پیاده‌سازی WireGuard ایکس‌ری پشت یک پروکسی SOCKS5 اختصاصی تست می‌کند
4. نتایج زنده را تماشا کنید — latency و وضعیت هر اندپوینت فوراً نمایش داده می‌شود
5. روی یک اندپوینت کلیک کنید تا کپی شود، سپس از **Apply Endpoint** برای به‌روزرسانی دسته‌جمعی چندین فایل کانفیگ استفاده کنید
6. کانفیگ‌های تغییر یافته در پوشه خروجی دلخواه شما ذخیره می‌شوند

## دریافت کانفیگ

برنامه شامل یک بخش راهنمای کامل با لینک‌های ژنراتورهای آنلاین، بات‌ها و کانال‌های تلگرام، ابزارهای خط فرمان و اپلیکیشن‌های کلاینت است.

## کامپایل از سورس

نیاز به **Go 1.26+** دارد (کامپایلر C لازم نیست):

```powershell
git clone https://github.com/QMahyar/Cloudflare-Scanner.git
cd Cloudflare-Scanner
go build -ldflags="-s -w" -o Cloudflare-Scanner.exe .
```

برای راهنمای کامل کامپایل، ساختار پروژه و مستندات معماری به [BUILD.fa.md](BUILD.fa.md) مراجعه کنید.

## استفاده

آخرین فایل `Cloudflare-Scanner-*.zip` را از [ریلیزها](https://github.com/QMahyar/Cloudflare-Scanner/releases) دانلود کنید، هردو فایل را extract کنید و `Cloudflare-Scanner.exe` را اجرا کنید. xray-core همراه آن است — نیازی به دانلود اضافی نیست.

برای بستن سرور، پنجره ترمینال را ببندید.
