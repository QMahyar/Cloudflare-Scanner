# Cloudflare Scanner

**اسکنر اندپوینت واپ و آی‌پی تمیز کلاودفلر** — یک ابزار دسکتاپ ویندوز با رابط کاربری وب. سه ابزار در یک: اندپوینت‌های working واپ کلاودفلر را پیدا می‌کند (وقتی ISP شما ترافیک استاندارد واپ را مسدود کرده)، محدوده آی‌پی‌های کلاودفلر را برای پروکسی‌های تمیز با لینک VLESS اسکن می‌کند، و آی‌پی:پورت را در کانفیگ‌های اشتراک با اندپوینت‌های تمیز جایگزین می‌کند.

> [English version](README.md)

## شروع سریع

1. آخرین فایل `Cloudflare-Scanner-*.zip` را از [ریلیزها](https://github.com/QMahyar/Cloudflare-Scanner/releases) دانلود کنید
2. هردو فایل را extract کنید و `Cloudflare-Scanner.exe` را اجرا کنید
3. یک برگه مرورگر در `http://127.0.0.1:XXXXX` باز می‌شود
4. برای بستن سرور، پنجره ترمینال را ببندید

xray-core نسخه v1.8.24 همراه برنامه است — نیازی به دانلود اضافی نیست.

## برگه‌ها

### اسکنر اندپوینت (Endpoint Scanner)

محدوده آی‌پی واپ کلاودفلر را اسکن می‌کند تا اندپوینت‌هایی که کانفیگ WireGuard شما روی آنها کار می‌کند پیدا کند. از xray-core پشت یک پروکسی SOCKS5 استفاده می‌کند، با نویز UDP اختیاری برای عبور از مسدودسازی DPI توسط ISP.

**نحوه استفاده:**
1. یک فایل کانفیگ WireGuard `.conf` انتخاب کنید (استاندارد واپ یا سبک هاگوارتز)
2. تعداد اسکن و ورژن آی‌پی را تنظیم کنید (IPv4 / IPv6 / هر دو)
3. در صورت مسدودسازی واپ توسط ISP، نویز UDP را فعال کنید
4. روی **Start Scan** کلیک کنید — نتایج زنده نمایش داده می‌شوند
5. روی یک اندپوینت کلیک کنید تا کپی شود، سپس از **Apply Endpoint** برای به‌روزرسانی دسته‌جمعی کانفیگ‌ها استفاده کنید
6. کانفیگ‌های تغییر یافته در پوشه خروجی ذخیره می‌شوند

### اسکنر آی‌پی (IP Scanner)

آی‌پی‌های کلاودفلر را که به TCP پاسخ می‌دهند پیدا می‌کند (فاز ۱) و در صورت تمایل آنها را با لینک VLESS شما از طریق xray-core اعتبارسنجی می‌کند (فاز ۲). برای پیدا کردن آی‌پی‌های پروکسی تمیز کلاودفلر برای ابزارهایی مثل v2ray، Nekobox، Sing-box و غیره مفید است.

**نحوه استفاده:**
1. یک لینک اشتراک VLESS وارد کنید (برای فاز ۲) یا خالی بگذارید برای فقط فاز ۱
2. گزینه **1-phase mode** را فعال کنید تا اعتبارسنجی xray رد شود و فقط پروب TCP انجام شود
3. تعداد اسکن و تعداد تست‌های فاز ۲ را تنظیم کنید
4. روی **Start Clean Scan** کلیک کنید
5. **فاز ۱** — پروب TCP به آی‌پی‌های تولید شده کلاودفلر، نتایج به صورت زنده به‌روز می‌شوند
6. **فاز ۲** — بهترین نتایج فاز ۱ از طریق xray-core با کانفیگ VLESS شما اعتبارسنجی می‌شوند
7. آی‌پی‌های working را به صورت لینک‌های VLESS خروجی بگیرید یا لیست خام IP را دانلود کنید

### جایگزین آی‌پی (IP Replacer)

یک لینک اشتراک (مثل `https://example.com/sub?token=...`) را دریافت می‌کند، کانفیگ‌هایی که فقط در IP:port متفاوت هستند را یکتا می‌کند، سپس تمام ترکیب‌های کانفیگ‌های یکتای شما با لیست اندپوینت‌های تمیز را می‌سازد. عالی برای به‌روز کردن یک اشتراک قدیمی با آی‌پی‌های تازه از اسکنر آی‌پی.

**نحوه استفاده:**
1. لینک اشتراک را وارد کنید و روی **Fetch** کلیک کنید
2. کانفیگ‌های یکتا (با حذف IP:port از هویت) با چک‌باکس نمایش داده می‌شوند
3. کانفیگ‌های مورد نظر را انتخاب یا لغو انتخاب کنید
4. اندپوینت‌ها را از نتایج اسکن آی‌پی تمیز بچسبانید (یا هر لیست IP:port، یک خط در هر)
5. روی **Generate Configs** کلیک کنید — تمام ترکیب‌ها ساخته می‌شوند
6. همه را در کلیپ‌بورد کپی کنید یا به صورت فایل متنی دانلود کنید

## ویژگی‌ها

- **سه ابزار یکپارچه** — اسکنر اندپوینت، اسکنر آی‌پی، جایگزین آی‌پی — در یک برنامه
- **نویز UDP** — padding تصادفی + تأخیر برای عبور از مسدودسازی DPI واپ
- **نتایج زنده** — اندپوینت‌ها هم‌زمان با قبول شدن هر تست نمایش داده می‌شوند
- **اسکن آی‌پی تمیز** — تولید آی‌پی بر اساس CIDR از محدوده‌های رسمی کلاودفلر (۲۵ CIDR IPv4 + ۹۱ CIDR IPv6)
- **پشتیبانی از اشتراک** — دریافت، یکتا کردن و جایگزینی دسته‌جمعی IP:port در کانفیگ‌های VLESS
- **اعمال دسته‌جمعی** — یک اندپوینت working را انتخاب کنید و به چندین فایل کانفیگ هم‌زمان اعمال کنید
- **رابط وب** — رابط کاربری تمیز تحت مرورگر که توسط خود برنامه سرو می‌شود، بدون نیاز به وابستگی اضافی
- **خودکفا** — یک `.exe` شامل xray-core، نیاز به تنظیمات اضافی ندارد
- **قابل لغو** — اسکن را هر لحظه می‌توانید متوقف یا ریست کنید
- **رابط دو زبانه** — انگلیسی و فارسی با قابلیت جابجایی فوری

## دریافت کانفیگ واپ

برنامه شامل یک بخش راهنمای کامل با لینک‌های ژنراتورهای آنلاین، بات‌ها و کانال‌های تلگرام، ابزارهای خط فرمان و اپلیکیشن‌های کلاینت است.

## کامپایل از سورس

نیاز به **Go 1.26+** دارد (کامپایلر C لازم نیست):

```powershell
git clone https://github.com/QMahyar/Cloudflare-Scanner.git
cd Cloudflare-Scanner
go build -ldflags="-s -w" -o Cloudflare-Scanner.exe .
```

برای راهنمای کامل کامپایل، ساختار پروژه و مستندات معماری به [BUILD.fa.md](BUILD.fa.md) مراجعه کنید.
