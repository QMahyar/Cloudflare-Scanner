# Cloudflare Scanner

**اسکنر اندپوینت واپ و آی‌پی تمیز کلاودفلر** — یک ابزار دسکتاپ کراس پلتفرم (ویندوز، لینوکس، مک، Termux/اندروید) با رابط کاربری وب. سه ابزار در یک: اندپوینت‌های working واپ کلاودفلر را پیدا می‌کند (وقتی ISP شما ترافیک استاندارد واپ را مسدود کرده)، محدوده آی‌پی‌های کلاودفلر را برای پروکسی‌های تمیز با لینک VLESS اسکن می‌کند، و آی‌پی:پورت را در کانفیگ‌های اشتراک با اندپوینت‌های تمیز جایگزین می‌کند.

> [English version](README.md)

## شروع سریع

1. آخرین فایل فشرده مخصوص پلتفرم خود را از [ریلیزها](https://github.com/QMahyar/Cloudflare-Scanner/releases) دانلود کنید
2. extract کنید و اجرا کنید:

   | پلتفرم | extract و اجرا |
   |---|---|
   | ویندوز | فایل `.tar.gz` را extract کنید (با 7-Zip یا `tar.exe`)، روی `Cloudflare-Scanner.exe` دوبار کلیک کنید |
   | لینوکس | `tar -xzf *.tar.gz && chmod +x Cloudflare-Scanner xray && ./Cloudflare-Scanner` |
   | مک | `tar -xzf *.tar.gz && chmod +x Cloudflare-Scanner xray && ./Cloudflare-Scanner` |
   | Termux | `tar -xzf *-termux-*.tar.gz && chmod +x Cloudflare-Scanner xray && ./Cloudflare-Scanner` |

3. یک برگه مرورگر در `http://127.0.0.1:XXXXX` باز می‌شود
4. برای بستن سرور، پنجره ترمینال را ببندید

xray-core نسخه v1.8.24 همراه برنامه است — نیازی به دانلود اضافی نیست.

## نحوه استفاده

برنامه شامل سه ابزار (برگه) است. در ادامه نحوه کار آنها را توضیح می‌دهیم:

```
اسکن اندپوینت واپ  ──>  انتخاب سریع‌ترین اندپوینت
                              │
اسکن آی‌پی تمیز    ──>  پیدا کردن آی‌پی‌های پروکسی working کلاودفلر
                              │
جایگزین آی‌پی       ──>  جایگزینی IP:port در کانفیگ‌ها با اندپوینت‌های تمیز
```

### اسکنر اندپوینت (Endpoint Scanner)

اندپوینت‌های واپ کلاودفلر را که کانفیگ WireGuard شما روی آنها کار می‌کند پیدا می‌کند. از xray-core پشت پروکسی SOCKS5 استفاده می‌کند، با نویز UDP اختیاری برای عبور از مسدودسازی DPI.

1. **انتخاب کانفیگ** — یک فایل WireGuard `.conf` انتخاب کنید (استاندارد یا هاگوارتز با S1-S4، Jc، Jmin، H1-H4، I1، I2)
2. **تنظیم تعداد اسکن** — سریع (۱۰۰)، معمولی (۵۰۰)، عمیق (۱۰۰۰)، شدید (۵۰۰۰)، حجیم (۱۰۰۰۰) یا دلخواه
3. **ورژن آی‌پی** — فقط IPv4 (پیش‌فرض)، فقط IPv6، یا هر دو
4. **نویز UDP** — در صورت مسدودسازی واپ توسط ISP فعال کنید
5. **شروع اسکن** — نتایج به صورت زنده نمایش داده می‌شوند. هر اندپوینت از طریق xray-core تست می‌شود
6. **اعمال اندپوینت** — روی اندپوینت کلیک کنید تا کپی شود، سپس از **Apply Endpoint** برای به‌روزرسانی دسته‌جمعی کانفیگ‌ها استفاده کنید

### اسکنر آی‌پی (IP Scanner)

آی‌پی‌های کلاودفلر را از محدوده‌های رسمی CIDR تولید می‌کند (۲۵ IPv4 + ۹۱ IPv6)، در فاز ۱ پروب TCP می‌کند، سپس در فاز ۲ بهترین‌ها را با لینک VLESS شما از طریق xray-core اعتبارسنجی می‌کند.

1. **ورود لینک VLESS** — یک لینک `vless://...` یا `trojan://...` برای فاز ۲ وارد کنید. برای فقط پروب TCP خالی بگذارید
2. **حالت ۱ فازه** — فعال کنید تا اعتبارسنجی xray رد شود و فقط پروب TCP انجام شود. پورت پیش‌فرض ۴۴۳
3. **تعداد اسکن** — تعداد آی‌پی‌هایی که تست می‌شوند
4. **تعداد فاز ۲** — چند تا از بهترین‌های فاز ۱ از طریق xray اعتبارسنجی شوند
5. **شروع اسکن تمیز**
6. **فاز ۱** (زنده) — اتصال TCP `net.DialTimeout` به هر IP:port با ۵۰۰ کارگر هم‌زمان. نتایج به صورت زنده نمایش داده می‌شوند
7. **فاز ۲** — مرتب شده بر اساس latency، N تا از بهترین‌ها از طریق xray-core و SOCKS5 با کانفیگ VLESS شما اعتبارسنجی می‌شوند. ۱۲ کارگر هم‌زمان
8. **خروجی** — دانلود آی‌پی‌های working به صورت لینک‌های VLESS یا لیست خام IP:port

### جایگزین آی‌پی (IP Replacer)

کانفیگ‌ها را (از لینک اشتراک یا چسباندن مستقیم) دریافت می‌کند، با اثر انگشت (نادیده گرفتن IP:port و remark) یکتا می‌کند، سپس تمام ترکیب‌ها را با لیست اندپوینت‌ها می‌سازد.

1. **انتخاب روش ورودی** — **Subscription URL** یا **Paste Configs** (فقط یکی فعال است)
   - لینک اشتراک: آدرس `https://example.com/sub?token=...` را وارد کنید، روی **Fetch** کلیک کنید
   - چسباندن کانفیگ: خطوط `vless://` و `trojan://` را بچسبانید، روی **Parse** کلیک کنید. خطوط اضافی نادیده گرفته می‌شوند
2. **بررسی کانفیگ‌های یکتا** — قالب‌های یکتا شده با چک‌باکس نمایش داده می‌شوند. انتخاب/لغو انتخاب کنید
3. **چسباندن اندپوینت‌ها** — یک `ip:port` در هر خط، از نتایج اسکن آی‌پی تمیز یا هر جای دیگر
4. **ساخت کانفیگ‌ها** — تمام ترکیب‌های کانفیگ × اندپوینت ساخته می‌شود. به remark هر کانفیگ ` @ endpoint` اضافه می‌شود
5. **کپی همه** یا **دانلود** خروجی

**نحوه تجزیه متن:** کانفیگ‌ها می‌توانند با فاصله، خط جدید، کاما، سمیکالن یا پایپ جدا شده باشند — همه کار می‌کنند. لینک‌های معتبر `vless://` و `trojan://` استخراج می‌شوند و بقیه نادیده گرفته می‌شوند.

## ویژگی‌ها

- **سه ابزار یکپارچه** — اسکنر اندپوینت، اسکنر آی‌پی، جایگزین آی‌پی — در یک برنامه
- **نویز UDP** — padding تصادفی + تأخیر برای عبور از مسدودسازی DPI واپ
- **نتایج زنده** — اندپوینت‌ها هم‌زمان با قبول شدن هر تست نمایش داده می‌شوند
- **اسکن آی‌پی تمیز** — تولید آی‌پی بر اساس CIDR از محدوده‌های رسمی کلاودفلر (۲۵ CIDR IPv4 + ۹۱ CIDR IPv6)
- **پشتیبانی از اشتراک** — دریافت، یکتا کردن و جایگزینی دسته‌جمعی IP:port در کانفیگ‌های VLESS
- **اعمال دسته‌جمعی** — یک اندپوینت working را انتخاب کنید و به چندین فایل کانفیگ هم‌زمان اعمال کنید
- **رابط وب** — رابط کاربری تمیز تحت مرورگر که توسط خود برنامه سرو می‌شود، بدون نیاز به وابستگی اضافی
- **خودکفا** — یک `.exe` شامل xray-core، نیاز به تنظیمات اضافی ندارد
- **قابل لغو** — اسکن را هر لحظه می‌توانید متوقف یا ریست کنید
- **رابط دو زبانه** — انگلیسی و فارسی با قابلیت جابجایی فوری

## دریافت کانفیگ واپ

برنامه شامل یک بخش راهنمای کامل با لینک‌های ژنراتورهای آنلاین، بات‌ها و کانال‌های تلگرام، ابزارهای خط فرمان و اپلیکیشن‌های کلاینت است.

## کامپایل از سورس

نیاز به **Go 1.26+** دارد (کامپایلر C لازم نیست). کراس پلتفرم — برای ویندوز، لینوکس و مک کامپایل می‌شود:

```bash
git clone https://github.com/QMahyar/Cloudflare-Scanner.git
cd Cloudflare-Scanner

# ویندوز
GOOS=windows GOARCH=amd64 go build -ldflags="-s -w" -o Cloudflare-Scanner.exe .

# لینوکس
GOOS=linux GOARCH=amd64 go build -ldflags="-s -w" -o Cloudflare-Scanner .

# مک (Apple Silicon)
GOOS=darwin GOARCH=arm64 go build -ldflags="-s -w" -o Cloudflare-Scanner .
```

برای راهنمای کامل کامپایل، نکات مختص پلتفرم، ساختار پروژه و مستندات معماری به [BUILD.fa.md](BUILD.fa.md) مراجعه کنید.
